# Discord-Bot
Перед тем, как запустить бота, создайте файл `credentials.json` с таким контентом:

```
{
  "discord-bot-token": "...",
  "discord-guild-id": "...",
  "openAI-api-key": "...",
  "cloudflare-api-key": "...",
  "cloudflare-account-id": "...",
  "cloudflare-namespace-events": "...",
  "cloudflare-namespace-schedule": "..."
}
```
